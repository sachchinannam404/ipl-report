# IPL Report Application

## Overview

This project is a web application that will display a report of IPL data. It is built using ASP.NET Core Razor Pages.

## Design

The application will have a clean and modern design, with a focus on usability and readability. The color scheme will be based on the IPL logo, and the typography will be clear and easy to read.

## Features

*   Display a list of IPL matches.
*   Allow users to filter matches by team and season.
*   Display detailed information about each match.

## Plan

*   Set up the basic project structure.
*   Create a model for the IPL data.
*   Create a service to read the IPL data from a CSV file.
*   Create a Razor Page to display the IPL data.
*   Style the Razor Page using a modern CSS framework.
